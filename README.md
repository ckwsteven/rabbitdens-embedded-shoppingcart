## rabbitdens embedded shoppingcart system https://rabbitdens.com/RDSCarousel/trunk/ui/lang/en_US/index.html
Easily embed the shopping system into your website and start selling goods in 10 minutes! Free, simple, and with PayPal support.

  ## Why Choose RabbitDens?
- **Fast**: RabbitDens includes a complete shopping and marketing system that can be used by simply pasting the code into your existing website.
- **Simple**: Embed it into your website in 3 steps, no technical background required.
- **Localized**: Supports LINE Pay and NewebPay, tailored for the Asian market.

## Installation Steps
1. Register for a RabbitDens account: https://www.rabbitdens.com
2. List your products on the backend and generate the embed code.
3. Paste the code into your website’s HTML and start selling immediately!

## Use Cases
With RabbitDens' embedded technology, it’s not just a regular website—you can also integrate multiple stores to create an online shopping street or mall:
- Sanxia Green Tea Season https://nas.rabbitdens.com/sanxia/index.html: A collective online marketplace for tea shops in Sanxia. Within the "Shopping Street," each store has its own independent shopping cart system. Additionally, product listings are "synchronized" and updated in the "Store Introduction" section, reflecting items from each store’s lightweight RabbitDens-generated website.
